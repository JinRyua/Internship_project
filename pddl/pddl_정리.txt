PDDL = Planning Domain Definition Language
=> 고전적인 플래닝 업무를 위한 표준 인코딩 언어.

PDDL 요소.
* Objects: 현실 세계의 것.
* Predicates: objects의 속성. true, false가 될 수 있음.
* Initial sate: 초기 상태.
* Goal specification: 목표 상태.
* Actions/Operators: 현실 상태를 변화시키는 방법.

PDDL File
* domain file: predicates와 actions.
* problem file: objects, initial sate, goal specification.
